# 🔐 VaultX - Password Manager & Generator

A fast, secure, and minimal password manager built with Flutter.

---

## ✨ Features

* 🔑 Secure password storage
* ⚡ Powerful password generator
* 📱 Clean & modern UI
* 💾 Local storage (offline support)
* 🎯 Simple and lightweight
